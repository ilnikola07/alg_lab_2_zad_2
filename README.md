# alg_lab_2_zad_2


**ФОРМУЛИРОВКА ЗАДАНИЯ**


Это 2 задание лаборатоной работы №2. Приложение написано на языке C# и с помощью Windows Forms реализует графический интерфейс для общения с пользователем. Оно принимает на вход число от пользователя (больше 0 и меньшее 100) и генерирует все нечётные числа до введённого пользователем числа и выводит их в ListBox. Также, как и в 1 задании лаборатоной работы №2, в приложении присутствует кнопка помощи и формулировки задания.


**КАК ЗАПУСТИТЬ**


Чтобы запустить это приложение, скачайте даннный репозиторий и запустите sln файл. Если всё прошло успешно - пользуйтесь рабочей формой в соответствии с инструкциеми (для этого в приложении реализована кнопка с помощью). В случае возникновения ошибок (такое увы бывает) просто перейдите и закомментируйте ошибочные 8 строк. Готово! От этого приложение не теряет своих способностей.
